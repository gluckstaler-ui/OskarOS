Design Directions — Three Visual Upgrades

Same six FalCaMel directions, three different visual approaches. The current card is a spec sheet — italic prose, mono font-names, thin palette strip. Reading mode. These three move the card to feeling mode.

Lever: amplify TYPE. Real fonts at real sizes. The user SEES Press Start 2P vs Sangbleu vs Playfair, not just the names. The font IS the founder (user.md).

Three approaches, same six directions. Pick the one that lets the user feel the difference fastest.
My pick: Proposal B (Moodboard). Image-led beats type-led beats lineage-led for a measurement instrument — image is the fastest path to reflex (vs evaluation). Cost is real but the 6 images become reusable wireframe assets, so the budget is shared.
