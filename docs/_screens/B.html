B. Record & detail screens — the deep views

B1 — Contact / Person Record

All · 360° on a person — identity, engagement, activity spine, related deals & people · ref: Attio, HubSpot, Folk

B1 · Contact Record — ① engagement meter ●●●●○ in the header ② timeline as the spine, not a tab ③ AI insight surfaces best-time-to-reach

B2 — Company / Account Record

AE, CS · Account 360° — org tree, contacts, open deals, tickets, signals feed, ★AI brief · ref: Salesforce, Attio

B2 · Account Record — ① org-hierarchy tree with per-unit ARR ② news/signals feed wired to next-best-action ③ ★AI account brief with risks + recommended step

B4 — Lead Record

SDR · Pre-qualification — score dial, source, activity, BANT checklist, prominent Convert CTA · ref: Salesforce, HubSpot

B4 · Lead Record — ① scored dial with weighted +/− factors ② BANT checklist, 3/4 met with the gap flagged ③ prominent Convert-to-deal CTA spells out what it creates

B5 — Activity Timeline (cross-record)

All · Full-screen unified reverse-chron feed with type filters and a left date rail — the spine as its own surface · ref: Attio, HubSpot

B5 · Activity Timeline — ① type-filter chips scope the feed ② sticky left date rail jumps to any day ③ every entry links back to its source record

B6 — Call / Conversation Detail

Manager, AE · Recording player with waveform, transcript, ★AI summary, tracker tags, moments & comments · ref: Gong, Chorus, Fathom

B6 · Call Detail — ① waveform scrubber with colored moment markers + tracker chips ② ★AI summary turns next-steps into auto-tasks ③ transcript highlights trackers inline (pricing/competitor)

B7 — Email / Message Detail

All · Full thread, right-rail open/click tracking, ★AI summary, draft-reply composer · ref: Outreach, Apollo

B7 · Email Detail — ① ★AI summary reads sentiment + status off the thread ② open/click tracking rail with timestamps (incl. the forward to the CFO) ③ AI-drafted reply with one-click Send / ★Rewrite

B8 — Task Detail

All · Focused single task — due date, priority, linked-record context, snooze/reassign/complete, subtasks · ref: HubSpot

B8 · Task Detail — ① AI-authored task carries due/priority and a description tied to the call ② subtask checklist with a progress bar ③ linked deal + contact context, so the task never floats free

B9 — Note / Document Editor

All · Rich-text editor — formatting toolbar with ★AI-rewrite, @mention, slash-command hint, attached file chip · ref: Attio

B9 · Note Editor — ① ★AI-rewrite button anchors the formatting toolbar ② live @mention chip + slash-command hint for blocks ③ attached-file chip lives inline in the doc body

B10 — Quote / Proposal Builder (CPQ)

AE · Line-item table, totals summary, discount-approval banner, e-signature block · ref: Salesforce CPQ, DealHub

B10 · Quote Builder — ① discount-approval banner fires when a line breaches the rep's limit, routed to a manager ② per-line discount % rolls into the totals summary ③ e-signature block awaiting the CFO

B11 — Deal Room (external)

AE + buyer · Buyer-facing shared space — mutual action plan, shared files, recap video · ref: Dock, DealHub

B11 · Deal Room — ① lighter external chrome + co-branded FunnelOS × Acme header ② mutual action plan with owners, due dates & the current step highlighted ③ personalized recap-video thumbnail over the shared files

B12 — Custom Object Record

Varies · The B1–B3 record shell for an org-specific object (Property) — custom-fields panel, timeline, related lists, Custom-object badge · ref: Salesforce, Attio

B12 · Custom Object Record — ① dashed "Custom object" badge marks an admin-defined type ② identical 3-column record shell as B1–B3 (custom fields · timeline spine · related lists) ③ related deals/contacts/docs adapt to the Property object
