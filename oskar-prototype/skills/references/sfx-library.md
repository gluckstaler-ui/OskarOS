## Usage Conventions

### Volume guidelines (from apple-gallery-showcase.md two-track audio)
- **SFX main track**: `1.0` (no attenuation)
- **BGM background track**: `0.4 ~ 0.5` (so SFX punches through clearly)
- **Multi-SFX stacking**: use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg combine templates
```bash
# Single SFX aligned to a timestamp:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multi-SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection decision tree
1. **Tactile action** (typing / clicking / sliding) → `keyboard/` or `ui/`
2. **Element entry / exit** → `transition/`
3. **Container-level operations** (card / modal) → `container/`
4. **State feedback** (success / error / notification) → `feedback/`
5. **Progress / time elapsed** → `progress/`
6. **Brand landing / key moment** → `impact/`
7. **AI magic / transform** → `magic/`
8. **Command-line / code demo** → `terminal/`

### Avoid pile-ups
- Same timestamp: max 2 concurrent SFX
- Drop BGM below 0.3 to allow 3 concurrent
- For brand impact moments, clear other SFX (leave 0.2s of silence before the land)

## Prompt Authoring Principles (for reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Three ingredients of a good prompt**:
1. **Physical sound description**: what object, what action ("mechanical keyboard single key press")
2. **Texture / style qualifier**: apple-style / ios-style / cinematic / retro
3. **Negative exclusions**: no reverb / clean studio / minimal

Bad: "click sound"
Good: "crisp ui button click, clean modern interface sound, apple-style, high pitched"

Bad: "magic"
Good: "bright twinkling stars sound, high pitched glittery chime, fairy dust"
