## ROOT CAUSE

**File:** `app/page.tsx`, lines 459-464

```javascript
// Auto-populate source images from session images (for new sessions)
useEffect(() => {
  if (sessionImages.length > 0 && sourceImages.length === 0) {
    setSourceImages(sessionImages.map(img => ({ ...img, selected: false })))
  }

```

**What this does:**
- When `sessionImages` changes AND `sourceImages` is empty → populate `sourceImages` with ALL session images

**The problem:**
1. Session starts → original images load → `sourceImages` gets populated ✓
2. User submits → `sourceImages` gets cleared (set to `[]`) 
3. Generated image arrives → `sessionImages` updates (now includes generated image)
4. `sourceImages.length === 0` is TRUE → **ALL session images (including generated) get re-added to sourceImages**

**Why Submit doesn't clear them:**
- Submit clears `sourceImages` to `[]`
- But then the `useEffect` IMMEDIATELY re-populates it because `sessionImages.length > 0 && sourceImages.length === 0`

## THE FIX

**Option A: Add a flag to track if user has submitted**

```javascript
const [hasSubmittedImages, setHasSubmittedImages] = useState(false)

// Auto-populate only on first load, not after submit
useEffect(() => {
  if (sessionImages.length > 0 && sourceImages.length === 0 && !hasSubmittedImages) {
    setSourceImages(sessionImages.map(img => ({ ...img, selected: false })))
  }
}, [sessionImages, hasSubmittedImages])

// In handleSubmitImages:
setHasSubmittedImages(true)
```

**Option B: Only auto-populate with ORIGINAL uploads, not generated images**

```javascript
useEffect(() => {
  if (sessionImages.length > 0 && sourceImages.length === 0) {
    // Only include images that are NOT generated (no 'shared-' prefix)
    const originalImages = sessionImages.filter(img => !img.filename.startsWith('shared-'))
    if (originalImages.length > 0) {
      setSourceImages(originalImages.map(img => ({ ...img, selected: false })))
    }
  }
}, [sessionImages])
```

**Option C: Track which images have been submitted**

```javascript
const [submittedImageIds, setSubmittedImageIds] = useState<Set<string>>(new Set())

useEffect(() => {
```
